IDE-360-Project-2

This repository is made for the purpose of sampling data from The 2025 Household Trends and Outlook Pulse Survey (HTOPS) Survey using R to create an ordinal logistic regression model to predict frequency of anxious feelings based on an individual's basic needs,  participation in art, difficulty meeting expenses, frequency of loneliness, ability to take care of themself, and their work status in the past two weeks.

It is important that the packages are installed. If they are not installed, uncomment the code lines beginning with 'install' before running. Comment the code back out to avoid errors when running. 

It is also important that the csv file maintain the same name to ensure it is properly imported for processing. 

The code will first perform exploratory analysis before training and running the model. Plots relevant to both phases of data exploration will be generated, along with summaries and further details printed to the terminal.
